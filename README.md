# Warehouse Robot Navigation

## Overview
A navigation system for autonomous warehouse robots. It enables robots to plan paths, avoid obstacles, and optimize movement in dynamic warehouse environments.

## Features
- Path planning using A* algorithm.
- Real-time obstacle detection and avoidance.
- Integration with ROS (Robot Operating System).
- Simulation support for testing in Gazebo.

## Requirements
- Tensorflow
- gymnasium
- Python 3.11+, `pandas`, `numpy`
